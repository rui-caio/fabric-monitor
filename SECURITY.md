# Security Policy

## Supported Versions

Currently, only the latest version of Fabric Monitor is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |

## Reporting a Vulnerability

We take the security of Fabric Monitor seriously. If you find a security vulnerability, please do NOT open a public issue. Instead, report it privately.

### How to report

Please send an email to <rui.caio@niw.pt>.

Include the following information:

- Description of the vulnerability.
- Steps to reproduce (proof of concept).
- Potential impact.

### Our Response

- We will acknowledge receipt of your report within 48 hours.
- We will work on a fix and keep you updated on our progress.
- Once fixed, we will publish a new version and, if appropriate, give you credit for the discovery.
